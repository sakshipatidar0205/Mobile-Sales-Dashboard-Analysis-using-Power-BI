# 📊 Mobile Sales Dashboard | Power BI Project

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze mobile sales performance across different cities, brands, payment methods, and time periods.

The dashboard provides valuable business insights such as:
- Total sales performance
- Month-to-date (MTD) analysis
- Same period last year comparison
- Sales trends by city and month
- Payment method analysis
- Brand-wise performance tracking

The project helps businesses make data-driven decisions using visual analytics.

---

# 🖥️ Dashboard Pages

## 1️⃣ Dashboard
Main overview page containing:
- Total Sales
- Total Quantity
- Average Price
- Transactions
- Sales by City
- Sales by Month
- Payment Method Analysis
- Brand Performance

## 2️⃣ MTD Report
Tracks Month-To-Date sales performance with:
- Daily sales trends
- Time-based analysis
- KPI monitoring

## 3️⃣ Same Period Last Year
Compares current sales performance with previous year data:
- Year-wise comparison
- Quarter-wise comparison
- Monthly comparison
- Growth analysis

# 📈 Key Insights

- Samsung generated the highest sales among brands.
- UPI and Credit Card were major payment methods.
- Sales performance increased steadily during the month.
- Certain cities contributed significantly higher revenue.
- Quarterly comparisons helped identify growth trends.
